# template-web-node
A template for building websites.

## Features / tools
* [dust][dustjs-linkedin]
* [express][]
* [jQuery][]
* [mongoDB][] using [node-mongodb-native][]

## Prerequisites
* [git][]
* [mongoDB][]
* [Node.js][]
* [npm][]

## Install & Run
    git clone https://github.com/zarac/template-web-node.git
    cd template-web-node
    npm install
    npm start
